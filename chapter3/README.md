## REPL
- 터미널, 프롬포트에서 자바스크립트 코드 입력
- 입력한 값의 결과값이 바로 출력됨.
- 하지만 간단한 코드를 테스트 하는 용도로 적합
- 긴 코드를 이용하기 힘듬.
- 그래서 파일을 작성해서 실행을 시키는 방법으로 진행
